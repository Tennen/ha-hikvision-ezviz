# EZVIZ HCNet SDK Backend Add-on

This add-on provides a glibc-based backend API for the `ezviz_hcnet` Home Assistant
custom integration.

- Architecture: `aarch64`
- API port: `8099`
- Default URL (with `host_network: true`): `http://127.0.0.1:8099`

The add-on includes bundled HCNetSDK arm64 libraries from the integration source tree.
